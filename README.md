# Femboy Chat - Regular & Freaky Mode

A cute dual-mode AI chatbot featuring a soft femboy in **Regular Mode** and a much lewder version in **Freaky Mode**.

Built as a single-page web app using vanilla HTML, CSS, and JavaScript. No backend or dependencies required.

## Features

- Toggle between **Regular Mode** (soft & wholesome) and **Freaky Mode** (explicit & keyword-reactive)
- Full keyword trigger system in Freaky Mode (`daddy`, `breed`, `punish`, `choke`, `slut`, `spank`, etc.)
- Cute typing indicator and smooth animations
- Fully responsive and mobile-friendly
- Works completely offline

## How to Run

1. Download or clone the entire `femboy-chat` folder
2. Open `index.html` in any modern web browser
3. No installation or server needed!

Use the toggle switch at the top to switch modes.  
In Freaky Mode, try typing keywords like `daddy`, `breed`, `punish`, `choke`, `slut`, `cute`, etc. for special responses.

## Customization

- Add/edit responses in `js/regular.js` or `js/freaky.js`
- Add new keyword triggers by extending the `keywordTriggers` object in `js/freaky.js`
